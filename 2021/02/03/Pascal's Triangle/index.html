Pascal's Triangle

Description:

Given an input parameter num_rows, generate the first num_rows number of rows of Pascal’s triangle.
As a quick refresher - in a Pascal’s triangle, each number is equal to the sum of the two directly above it.

Example:

This problem is quite like the Dynamic Programming method in the problem Count paths on a game board. It is not difficult to solve.

The keypoint is to initialize the result vector. I did not initialize the vector at beginning, I just push the vectors while I am creating it, which the space complexity S(n) is n. However, the other way is to initialize the vector at beginning, which the S(n) is 2n.
