Find the Number that Appears Once

Description:

Write a method that returns a number that appears only once in an array.
Assume the array will surely have a unique value. If the array is empty return 0.

Examples:

{1,2,3,4,1,2,4,3,5} ==> 5

Firstly, sort the input array. Then create a int tmp to store current compare target and a int dup to flag whether current element duplicated. In the end, return the tmp as the answer.
